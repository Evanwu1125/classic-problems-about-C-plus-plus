字符串的左旋转操作是把字符串前面的若干个字符转移到字符串的尾部。

请定义一个函数实现字符串左旋转操作的功能。

比如输入字符串"abcdefg"和数字 2
，该函数将返回左旋转 2
 位得到的结果"cdefgab"。

### 注意：

数据保证 n
 小于等于输入字符串的长度。
### 数据范围
输入字符串长度 [0,1000]
。

### 样例
输入："abcdefg" , n=2  

输出："cdefgab"

```c++
class Solution {
public:
    string leftRotateString(string str, int n) {
        string tem;
        for (int i=n;i<str.size();i++) tem += str[i];
        for (int i=0;i<n;i++) tem += str[i];
        
        return tem;
    }
};
```
