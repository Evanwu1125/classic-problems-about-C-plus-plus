输入两个链表，找出它们的第一个公共结点。

当不存在公共节点时，返回空节点。

### 数据范围
链表长度 [1,2000]
。
保证两个链表不完全相同，即两链表的头结点不相同。

### 样例
给出两个链表如下所示：  
A：        a1 → a2  
                   ↘  
                     c1 → c2 → c3  
                   ↗              
B:     b1 → b2 → b3  

输出第一个公共节点c1

```c++
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
public:
    ListNode *findFirstCommonNode(ListNode *headA, ListNode *headB) {
        auto p = headA, q = headB;
        
        while (p != q)
        {
            if (p) p = p -> next;
            else p = headB;
            if (q) q = q -> next;
            else q = headA;
        }
        return p;
    }
};
```
